# Agent-ready project template

This is a template repository for **any type of project** that you want to build using agents.
It has a minimal set of instructions for any agent (OpenAI Codex, Claude Code, etc). The agent will then help you set up everything.

## Getting started
1. Install the coding agent of your choice.
   - OpenAI Codex: https://openai.com/codex/
   - Claude Code: https://code.claude.com/docs/en/overview
   - Factory.AI Droid: https://factory.ai/product/ide
   - Opencode: https://opencode.ai/
2. Clone and open this repository.
3. Start your coding agent and prompt it with "Help me get started!"
